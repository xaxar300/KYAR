Списки свойств HTML и CSS

1. Свойства для работы с текстом (CSS):

font-family: задает семейство шрифта.

font-size: определяет размер шрифта.

font-weight: устанавливает насыщенность шрифта (жирность).

line-height: задает межстрочное расстояние.

text-align: выравнивает текст по горизонтали (слева, справа, по центру, по ширине).

text-transform: меняет регистр текста (например, все заглавные или прописные).

color: определяет цвет текста.

2. Свойства для работы с блоками и размерами (CSS):

width: задает ширину элемента.

height: задает высоту элемента.

padding: внутренний отступ внутри элемента от содержимого до границы.

margin: внешний отступ элемента от других элементов.

border: задает границу элемента (толщина, стиль, цвет).

display: определяет тип отображения элемента (например, block, inline, flex).

box-shadow: добавляет тень вокруг элемента.

3. HTML-атрибуты для управления формами и вводом данных:

type: определяет тип поля ввода (text, email, password, и т.д.).

placeholder: текст-заполнитель, который отображается до ввода данных.

required: делает поле обязательным для заполнения.

maxlength: устанавливает максимальное количество символов для ввода.

min и max: задают минимальные и максимальные значения для числовых полей.
