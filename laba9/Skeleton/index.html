Применение CSS Grid Layout
Итоги

CSS Grid Layout (или просто Grid) — это метод для работы с двумерными макетами. В отличие от Flexbox, который ориентирован только на однонаправленную раскладку, Grid позволяет управлять как строками, так и столбцами на одной странице.

Grid предоставляет возможность одновременно контролировать расположение элементов как по вертикали, так и по горизонтали, что дает большую гибкость в создании сложных макетов.

grid-template-columns / grid-template-rows

Кроме привычных единиц измерения в CSS, таких как px и %, спецификация Grid вводит новую единицу — 1fr (fraction). Она аналогична свойству flex-grow в Flexbox, определяющему пропорции для растягивания элементов. В приведённом примере первая и вторая колонки имеют ширину 150px и 20vw соответственно, а третья колонка автоматически подстраивается под содержимое (с помощью значения auto). Использование 1fr позволяет заполнять оставшееся пространство.

Адаптивная сетка карточек с auto-fit

Примером может служить адаптивная сетка карточек с минимальной шириной 240px. Карточки будут растягиваться, чтобы заполнить всю доступную ширину родительского элемента. Это избавляет от необходимости вручную изменять сетку для разных размеров экрана — Grid сделает всё автоматически.

Grid — это мощный инструмент, который значительно упрощает создание макетов, для которых раньше требовались сложные решения. Он предлагает множество новых свойств и методов, расширяя возможности CSS. Хотя освоение всех возможностей Grid может быть сложной задачей, не обязательно учить и использовать их все сразу.

Чтобы начать работать с Grid, достаточно выбрать один из подходов, которые вам были показаны.
